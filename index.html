Beyond the Tavern

Your DnD Character Creator & Initiative Tracker

Empowering Your DnD Adventures

Welcome to Beyond the Tavern, your one-stop shop for crafting compelling DnD characters and managing initiative with ease! This enhanced version incorporates a hybrid approach for data access, leveraging both the online D&D 5e API (dnd5eapi.co) and optional offline data storage. Built with the powerful .NET MAUI framework, Beyond the Tavern caters to both online and offline DnD enthusiasts.

Core Functionalities (Initial Iteration)

  • **Character Creation:** Breathe life into your DnD hero! Select your character's race and class, meticulously assign ability scores, and choose a striking portrait image from your device storage. A dedicated page for managing spells and cantrips ensures your character is fully equipped for any adventure.
  • **Initiative Tracker:** Streamline your DnD sessions with our intuitive initiative tracker. Players and Dungeon Masters can effortlessly add characters with names and initiative modifiers. Witness the initiative order automatically sorted (highest first) on a separate page, ensuring a smooth and exciting gameplay flow.

Unveiling the Power of Choice: Hybrid Data Access

Beyond the Tavern offers a flexible approach to data access, empowering you to choose what best suits your needs.

Online Mode (Default)

Leverage the convenience of online access! Beyond the Tavern seamlessly integrates with the official D&D 5e API (dnd5eapi.co), granting you access to a vast repository of DnD information. Experience auto-completion suggestions during character creation, delve into detailed descriptions of races, classes, and spells, and potentially explore integrated spell descriptions (subject to API limitations).

Offline Mode (Optional)

Don't let a lack of internet connection hinder your DnD experience! Beyond the Tavern empowers you with optional offline functionality. Download a local data package containing relevant D&D 5e information and access it seamlessly when an internet connection is unavailable.

Benefits

  • **Convenience (Online Mode):** Online access provides ease of use and the potential for automatic updates, keeping your D&D data fresh.
  • **Offline Functionality:** Never be without your DnD tools! Create characters, manage initiative, and access vital information even without an internet connection.
  • **Customization (Optional - Offline Mode):** The offline data approach offers the potential for modification or extension (consider copyright implications), allowing you to tailor your D&D experience further.
  • **Flexibility:** Choose the data access mode that best suits your needs, online or offline, the decision is yours!

Technical Details (For Developers)

Beyond the Tavern is built with the .NET MAUI framework, enabling it to run seamlessly on both Android and Windows devices. Here's a glimpse into the development considerations:

  • API Integration (Online Mode): Utilizes libraries like System.Net.Http to make HTTP requests to the D&D 5e API and parses JSON responses for data retrieval.
  • Offline Mode (Optional): Implements logic for downloading and storing a local data package, potentially using SQLite or another lightweight database for efficient storage and retrieval.
  • Error Handling: Proper error handling is implemented to address potential network connectivity issues and API calls.
  • User Interface: A user-friendly interface is designed leveraging the MVVM (Model-View-ViewModel) pattern for code organization and maintainability. Data binding is utilized for easier interaction between UI elements and underlying data models. UI optimizations are considered for different screen sizes and resolutions on both Android and Windows devices.
